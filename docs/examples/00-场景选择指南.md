## 阅读约定

与 [操作说明](../operation-guide.zh-CN.md) 一致：

- **`$ hx ...`**：在终端执行（管控面、批准、归档）。
- **`Cursor ▸`**：在 Cursor Agent 对话框执行（写规格、写代码、自校正）。
- 须先 `hx adapter sync`（场景 01）再使用斜杠命令。

完整索引见 [README](README.md)。
